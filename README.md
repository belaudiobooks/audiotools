# Audiotools

This repo contains command line tools for processing audio files.

## Usage

```
python -m venv ./venv
source ./venv/bin/activate
pip install -r requirements.txt
python script.py --help
```
## Development

Make sure you you have pre-commit hooks installed:

```
python -m venv ./venv
source ./venv/bin/activate
pip install -r requirements.txt
pre-commit install
```
